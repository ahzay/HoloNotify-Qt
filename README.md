# HoloNotify-Qt
Desktop tool for Hololive notifications and more in C++/Qt. A short demo can be seen below.

![demogif](demo.gif)

It's KDE theming friendly!

![demopng](demo.png)

## Compiling
Simply run qmake on the .pro file to generate a MakeFile then make.
